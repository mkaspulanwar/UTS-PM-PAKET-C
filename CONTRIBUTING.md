# Contributing Guidelines

Terima kasih telah tertarik untuk berkontribusi pada proyek ini!   
Berikut langkah-langkah sederhana untuk berkontribusi:

1. Fork repositori ini ke akun GitHub Anda.  
2. Clone hasil fork ke komputer lokal:
   git clone https://github.com/<username>/uts_mp_230104040212.git
3. Buat branch baru untuk perubahan Anda:
   git checkout -b fitur-baru-anda
4. Lakukan perubahan dan commit:
   git commit -m "Menambahkan fitur baru"
5. Push branch ke repository fork Anda:
   git push origin fitur-baru-anda
6. Buka Pull Request (PR) ke branch main di repository utama.

Aturan Dasar:
- Pastikan kode tetap bersih dan terformat dengan baik.
- Gunakan bahasa Indonesia atau Inggris secara konsisten di komentar.
- Jelaskan perubahan Anda dengan singkat di deskripsi PR.

Setiap kontribusi, sekecil apa pun, sangat berarti!
